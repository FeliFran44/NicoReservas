# Ipora Futbol Club - Sitio Web Oficial

🌐 **Landing Page oficial de Ipora Futbol Club**

## 📋 Descripción

Sitio web de Ipora Futbol Club, escuela de fútbol infantil ubicada en Maldonado, Uruguay. Fundada el 3 de Abril de 2018.

### Servicios
- 🏫 Escuela de Fútbol Infantil (3-13 años)
- ⚽ Cancha de Fútbol 5 (Reservas online)
- 💪 Fútbol Funcional
- 🎉 Salón de Fiestas

## 🚀 Tecnologías

- HTML5
- CSS3 (Variables CSS, Grid, Flexbox)
- JavaScript Vanilla
- Google Calendar API (Sistema de reservas)

## 📱 Características

- ✅ Diseño Responsive (Mobile First)
- ✅ Animaciones Scroll Reveal
- ✅ Contadores Animados
- ✅ Integración WhatsApp
- ✅ Sistema de Reservas Online
- ✅ Galería de Imágenes con Lightbox
